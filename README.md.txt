# UniUD StudySwap

Una piattaforma web per scambiare, vendere e acquistare appunti e libri universitari.

## Tecnologie
- Next.js
- Supabase
- TailwindCSS
- Stripe (in arrivo per i pagamenti)

## Sviluppo locale
1. Clona il repository
2. Crea il file `.env.local` con le tue chiavi Supabase
3. Installa le dipendenze:
   ```bash
   npm install
